sgnz-chat Web
====

## Overview
sgnz-chat web client

## Installation

```bash
$ npm install
```

## Quick Start

```bash
$ npm install
```

  Start the server:

```bash
$ npm start
```
## Build

  Development:

```bash
$ npm run postinstall:development
```

  Production:

```bash
$ npm run postinstall:production
```

## Git Update
```bash
$ git submodule foreach 'git checkout master; git pull'
$ git pull
```

## Deploy

```bash
$ sh deploy.sh
```
